
# 餘熱回收鍋爐 (HRSG) 設備安裝順序

## 一般注意事項

本文件（包含所有附件及引用的文件）說明森霸二期專案 (60031) 的餘熱鍋爐 (HRSG) 安裝順序。

部分描述的安裝步驟可同時進行，取決於可用的人力與機具。西門子能源 (Siemens HTT) 現場技術指導 (TFA) 可在現場針對安裝順序的任何細節或疑問提供建議。例如：進氣風道和煙囪的安裝可在模組安裝完成後開始進行。管線安裝與平台安裝可同時進行。

及時安裝樓梯塔有助於在餘熱鍋爐安裝期間方便進出，建議承包商檢查樓梯塔的安裝步驟與餘熱鍋爐安裝的配合情況，以盡可能避免額外工作或延誤。

為了提供清晰明瞭的概觀，無論是外殼/受壓組件，還是進氣風道和排氣風道/煙囪，各個組件的相應步驟都已大致歸類在一起。

然而，取決於可用的人力、機具和工作區域，平行作業是可行的。可向技術指導 (TFA) 諮詢有關執行平行作業的建議。

建議在早期階段（與外殼/受壓組件的最後收尾步驟平行）開始進氣風道的安裝。原因是及早安裝進氣風道有助於後續重要步驟的進行，例如進氣風道上方的管線與平台安裝，以及進氣風道與 Box 1 之間的主管線集管與導流板安裝。

開始進氣風道安裝活動的最佳時機，將由承包商與 TFA 根據現場實際情況進行確認。較合理的開始時機是當外殼成為一個完整的（結構）單元時。

在整個安裝階段，必須進行適當的油漆補漆與現場塗裝，包含適當的表面處理。補漆與現場塗裝不在 Siemens-Energy-HTT 的範圍內。

在整個安裝階段，必須遵守現場品質計畫 (FQP) 的要求。

### 豎琴式管排模組到達順序 – 現場吊裝活動

森霸二期專案的豎琴式管排模組供應狀況取決於工廠的交貨進度。在製作這些順序圖時掌握的最新資訊顯示，Box 6 的模組將最先抵達現場。

其餘的 Box 的模組將從餘熱鍋爐煙囪側朝進氣風道側依序抵達：

- Box 5 => Box 4 => Box 3 => Box 2 最後為 Box 1
    

為了不延誤現場工程，建議根據模組的可用性（從工廠運達的順序）依序進行豎琴式管排模組的安裝作業。

每個豎琴式管排模組請遵循步驟 26 至 33 進行安裝。

每個 Box 內的建議安裝順序為先安裝外部模組（左側與右側，靠近外殼側），隨後安裝中間模組。

若模組抵達順序與上述說明的順序不同，請諮詢 HTT TFA 以探討相應修改吊裝順序的可能性。

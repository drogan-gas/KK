依據手冊內容，這份表格彙整了 HRSG 各安裝步驟的施工方式、注意事項、QC 檢查項目與要求，以及對應的 QC 文件編號。

### HRSG 機械安裝步驟與 QC 要求彙整表

| 安裝步驟                                    | 安裝方式與注意事項                                             | QC 檢查項目與要求                                    | QC 文件名稱/編號                                 |
| --------------------------------------- | ----------------------------------------------------- | --------------------------------------------- | ------------------------------------------ |
| **基礎審查 (Foundation Review)**            | 使用第三方測量報告核實位置與標高；地栓口袋孔需清潔；接地需完成。                      | 核實實際尺寸、標高與地栓位置；基礎打毛 (Chipping) 需在鋼構安裝前完成。     | FQP - General (60031-730-10-001)           |
| **主鋼構預組裝 (Main Steel Pre-assembly)**    | 在水平位置組裝門架，使用臨時角鐵支撐；安裝橫向緩衝支撐。                          | 支撐需完全水平以防變形；禁止用火切擴孔；吊裝前完成最終檢查。                | FQP - Main Steel (60031-730-10-002)        |
| **主鋼構安裝 (Main Steel Erection)**         | 設定水平墊片 (Shim stacks)；使用吊帶吊裝；使用斜撐或纜繩 (Guy wires) 保持穩定。 | 檢查螺栓扭矩值；垂直度與對心需符合公差圖要求；對心時纜繩需放鬆。              | FQP - Main Steel (60031-730-10-002)        |
| **灌漿施工 (Grouting)**                     | 用於底板與支撐板下方；確保固化時間 (通常 5 天)。                           | 測試灌漿強度；檢查模板與施工面清潔度；灌漿需完全填滿無空洞。                | FQP - General (60031-730-10-001)           |
| **外殼安裝 (Boiler Casing)**                | 外部全連續焊，內部間斷焊；利用膠合板保護底板襯板。                             | 焊接前檢查對心與標高；檢查焊接氣密性；所有角落需用填充板密封焊。              | FQP - Boiler Casing (60031-730-10-003)     |
| **內部保溫與襯板 (Internal Insulation/Liner)** | 焊接保溫銷；保溫層需交錯排列且無空洞；襯板重疊需符合圖面。                         | 焊接前清潔表面；檢查螺母緊固度後再行點焊；保溫材料需保持乾燥。               | FQP - Boiler Casing (60031-730-10-003)     |
| **模組安裝 (Module Erection)**              | 使用專用吊梁與強背架 (Strong back)；由頂部載入；安裝後鎖定模組。               | 核實模組編號與位置；檢查懸吊桿對心標高；確認與橫向緩衝器的間隙。              | FQP - Modules (60031-730-10-004)           |
| **汽鼓安裝 (Drum Erection)**                | 安裝滑動支座與 PTFE 墊片；使用上升管輔助對心；安裝中心固定叉。                    | 檢查汽鼓標高與水平度；確保配管連接處無應力 (Stress-free)；檢查滑動面無污損。 | FQP - Drums (60031-730-10-006)             |
| **擋板與密封板 (Baffles & Sealing)**          | 分為閣樓 (Attic) 與地下室 (Basement) 擋板；安裝膨脹滑動點。              | 核實熱膨脹間隙 (Gap A)；長孔螺栓連接需手指緊固並點焊以維持活動。          | FQP - Baffles (60031-730-10-005)           |
| **煙囪安裝 (Stack Erection)**               | 基座環先置於基礎墊片；逐段吊裝並用纜繩固定；預組裝煙囪段以減少高空作業。                  | 檢查每段的圓度與垂直度；核實縱向焊縫參考點 (0/120/240度)；擋板試運轉。     | FQP - Stack (60031-730-10-009)             |
| **進氣/出氣煙道 (Inlet/Outlet Duct)**         | 進氣端使用可調法蘭補償公差；出氣端安裝消音器；全連續焊施工。                        | 檢查法蘭對心與螺栓孔一致性；核實消音器安裝位置與支撐；焊接氣密檢查。            | FQP - Inlet (10-011) / Outlet (10-010)     |
| **配管安裝 (Piping)**                       | 大管從系統固定點開始安裝；小管需預留 2mm 承插間隙；Grade 91 需嚴格控管焊接程序。       | 焊接前檢查坡口；查驗材料熱號 (Heat No.) 與清淨度；檢查坡度與支吊架位置。    | FQP - Piping (60031-730-10-008)            |
| **膨脹接頭 (Expansion Joints)**             | 建議由原廠專家指導；安裝前檢查法蘭無尖銳邊緣；填寫足夠保溫棉。                       | 核實螺栓扭矩值；確保保溫層厚度防止熱點 (Hot spots)；安裝後用焊毯保護。     | FQP - Expansion Bellows (60031-730-10-013) |
| **水壓測試 (Hydrotesting)**                 | 分系統進行；準備測試包 (Test Package)；由授權檢驗員核准。                  | 目視檢查焊道與配件；測試後務必移除安全閥的水壓堵頭。                    | Hydro testing procedure (60031-735-01)     |
| **SCR 與 AIG 系統**                        | 安裝支撐架與噴氨管路；催化劑模塊通常於初火後安裝。                             | 檢查噴嘴位置與標高；驗證內部支架對心。                           | FQP - SCR - AIG System (60031-730-10-014)  |

### QC 檢查之共通要求：

1.  **目視檢查 (VT)**：所有非壓力零件焊接需依據 AWS D1.1。
2.  **非破壞檢測 (NDE)**：壓力零件需符合 ASME 代碼要求。
3.  **螺栓緊固**：必須依據 structural steel (60031-730-01-008) 或 flanged connections (60031-400-20) 規範執行。
4.  **竣工紀錄 (As-Built)**：施工承包商需在圖面上紀錄所有偏差與實際狀態。

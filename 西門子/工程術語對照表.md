根據《台電 熱回收鍋爐主設備安裝工程分項施工暨品質計畫.md》與《HRSG 英文New.md》兩份來源文件，彙整出的工程術語對照表如下。本表區分為組織與職掌、法規與品質管理、以及設備與施工技術三大類別：

### 1. 組織職稱與通用術語

| 英文術語 / 縮寫                             | 中文術語           | 說明 / 來源                                   |
| :------------------------------------ | :------------- | :---------------------------------------- |
| **EPC / Client**                      | 統包商 / 業主       | 指 Siemens Energy 或 台電 (TPC)               |
| **Manufacturer**                      | 製造商            | 指 Siemens Energy B.V.                     |
| **Subcontractor**                     | 協力廠商           | 指負責安裝的廠商（如 CTCI 中鼎）                       |
| **AI (Authorized Inspector)**         | 公證檢查員          | 負責查核材料、銲接與試壓程序是否符合規範                      |
| **CWI (Certified Welding Inspector)** | 銲接檢驗師          | 負責銲接程序、銲工管理及現場銲道目視檢查                      |
| **HSE**                               | 安衛環            | 健康、安全與環境 (Health, Safety and Environment) |
| **WMS (Work Method Statement)**       | 工作方法說明書 (施工計畫) | 包含詳細施工步驟與吊裝計畫的文件                          |

### 2. 標準、規範與品質管理

| 英文術語 / 縮寫                               | 中文術語                | 說明 / 來源                 |
| :-------------------------------------- | :------------------ | :---------------------- |
| **ASME**                                | 美國機械工程師學會           | 鍋爐與壓力容器遵循之設計與建造標準       |
| **AWS**                                 | 美國銲接學會              | 非壓力件銲接遵循之標準             |
| **FQP (Field Quality Plan)**            | 現場品質計畫 (品質管理標準)     | 規定現場檢查項目、時機與頻率之標準       |
| **ITP (Inspection and Test Plan)**      | 檢驗與試驗計畫             | 施工品質控管的關鍵點清單            |
| **WPS / PQR / WPQ**                     | 銲接程序書 / 程序檢定 / 銲工資格 | 銲接作業必備的三大核准文件           |
| **NDE (Non Destructive Examination)**   | 非破壞檢測               | 包含 RT, UT, MT, PT 等檢測方法 |
| **PWHT (Post Weld Heat Treatment)**     | 銲後熱處理               | 銲接完成後消除應力的熱處理程序         |
| **MRR (Material Receiving Receipt)**    | 材料收料單               | 材料進場點收的紀錄憑證             |
| **OSD Report**                          | 溢領、短缺與損壞報告          | 材料接收發現偏差時提交的報告          |
| **LISL (Line Inspection Summary List)** | 管線檢驗彙整表             | 記錄管線材料熱號、銲接與 NDE 結果的總表  |

### 3. 設備組件與施工技術

| 英文術語 / 縮寫                     | 中文術語        | 說明 / 來源                               |
| :---------------------------- | :---------- | :------------------------------------ |
| **HRSG**                      | 熱回收鍋爐       | Heat Recovery Steam Generator         |
| **Casing**                    | 爐殼          | 包含底部 (Bottom)、側面 (Side) 與頂部 (Roof) 面板 |
| **Module / Harp**             | 管排模組        | 鍋爐核心換熱組件，分為 OCC (立式) 與 OT (臥式)        |
| **Steam Drum**                | 汽鼓          | 包含高壓 (HP)、中壓 (IP) 與低壓 (LP) 汽鼓         |
| **Separator Vessel**          | 分離器         | 高壓 section 的 Once through Benson 系統組件 |
| **Inlet / Outlet Duct**       | 進出口煙道       | 引導燃氣進入及離開鍋爐的通道                        |
| **Stack / Chimney**           | 煙囪          | 排放廢氣的最後端結構，含擋板 (Damper)               |
| **Expansion Joint / Bellows** | 膨脹接頭 / 伸縮接頭 | 補償熱膨脹位移的連接組件                          |
| **Baffle / Gas Baffle**       | 導流板 / 氣體導流板 | 導引氣流並分隔空間之金屬板                         |
| **Grouting**                  | 灌漿          | 於設備底座與基礎間填充無收縮水泥                      |
| **Shim stacks**               | 墊片組         | 用於調整鋼構或設備高程與水平的薄鐵片                    |
| **Strong back**               | 強背架         | 吊裝模組時用於增加剛性及翻轉的專用框架                   |
| **Strand Jack**               | 鋼索式液壓千斤頂    | 用於 OT 模組等重件的提升設備                      |
| **Hydrostatic Test**          | 水壓試驗        | 驗證壓力零件銲接完整性的最終測試                      |
| **Chipping**                  | 打毛          | 混凝土基礎面預處理以增加灌漿結合力                     |
